>>Marketplace - Azure Blockchain Workbench 

"https://blockchain-dodutl.azurewebsites.net/applications/1/workflows/1"

	-Project is deployed to Azure.

	-There are owners and buyers.

	-Owners puts items and set name and price. 
	-Transaction state is "Item Available".

	-Buyers bid and then the state of transaction changes to "Offer Pending".

	-Then owner can take 2 actions: 
    
		-Accept Offer
    
		-Reject Offer
	
-If owner accepts the offer the state of transaction becames "Sold".

	-Otherwise the state of transaction becomes "Item Available".
